TicketCheckerBot

Бот для поиска транспорта.

Основные возможности:

  • Поиск билетов
  • Поиск билетов с указанием вида транспорта
  • Поиск ближайшего транспорта
  • Поиск транспорта по времени отправления
  • Поиск по цене

Ключевые слова:

  • 'на' время (для указания времени отправления)
  • 'за','стоимостью','имея' количество денежных единиц (для указания стоимости билета)
  • 'ближайший' (для поиска ближайшего транспорта)
  • 'маршрутка'/'автобус', 'поезд'/'электричка' (для поиска соответствуюшего транспорта)
