TicketCheckerBot

Бот способен искать билеты на разные типы транспорта по заданному маршруту.

Основные возможности:

  • Поиск билетов
  • Поиск билетов с указанием вида транспорта
  • Поиск ближайшего транспорта
  • Поиск транспорта по времени отправления
  • Поиск по цене

Ключевые слова:

  • 'из' место1 'в' место2(для задания маршрута)
  • 'на' время (для указания времени отправления)
